About This Project

Project Overview
AI-Based Road Accident Detection Using CCTV (No Hardware)

This system is a sophisticated artificial intelligence application designed to automatically detect and report road accidents from CCTV footage. The system uses advanced deep learning models and computer vision techniques to analyze video frames in real-time and identify potential traffic incidents.

Key Features:

  • Vehicle Detection: Uses YOLOv5 for real-time vehicle detection
  • Accident Analysis: Detects vehicle collisions, overlaps, and abnormal motion
  • Multi-Frame Confirmation: Analyzes multiple frames to reduce false positives
  • Alert System: Generates immediate alerts when accidents are confirmed
  • Web Dashboard: User-friendly interface for viewing and managing detected incidents
  • Database Storage: Persistent storage of all accident records with metadata

How It Works
  1. Video Upload:
    User uploads a CCTV or traffic video through the web interface
  2. Frame Extraction:
    OpenCV extracts individual frames from the video at specified intervals
  3. Vehicle Detection:
    YOLOv5 model detects all vehicles in each frame with bounding boxes
  4. Motion Analysis:
    System analyzes vehicle movement and detects sudden motion changes
  5. Collision Detection:
    Overlapping bounding boxes and close proximity are analyzed for potential collisions
  6. Multi-Frame Confirmation:
    System analyzes multiple consecutive frames to confirm accidents and reduce false positives
  7. Alert Generation:
    When an accident is confirmed, an alert is generated with confidence score and metadata
  8. Dashboard Display:
    Confirmed accidents are displayed on the web dashboard with images and details
Accident Detection Logic

The system detects accidents using a combination of factors:

  • Vehicle Overlap Detection: Calculates Intersection over Union (IoU) between vehicle bounding boxes. High overlap indicates potential collision.
  • Motion Analysis: Detects sudden changes in vehicle positions between consecutive frames. Abnormal motion patterns may indicate accidents.
  • Confidence Scoring: Combines overlap score (70% weight) and motion score (30% weight) to calculate overall accident confidence.
  • Multi-Frame Confirmation: Requires at least 40% of buffered frames to show accident indicators for confirmation, reducing false positives.
